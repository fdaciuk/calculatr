# calculatr

> CLI Calculator

[![Build Status][travis-image]][travis-repo]

## How to use

### `calc <number1> <operator> <number2>`

Make a calc passing an operator:

```sh
calculatr calc 1 + 2
```

It returns `3`.

```sh
calculatr calc 4 - 3
```

It returns `1`.

```sh
calculatr calc 4 * 3
```

It returns `12`.

```sh
calculatr calc 10 / 2
```

It returns `5`.
